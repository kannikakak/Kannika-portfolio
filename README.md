# Portfolio Project

## Overview
This portfolio project showcases the skills and achievements of Kannika Kak, a full-stack web developer. The project is designed to present a professional online presence, highlighting various projects, skills, and volunteer experiences.

## Features
- **Responsive Design**: The portfolio is fully responsive, ensuring a seamless experience on both desktop and mobile devices.
- **Smooth Scrolling**: Navigation links allow for smooth scrolling to different sections of the page.
- **Project Filtering**: Users can filter projects by category (Web Development, Mobile Development, Data Science).
- **Contact Form**: A contact form is included for potential employers or collaborators to reach out.

## Installation Instructions
1. Clone the repository:
   ```
   git clone https://github.com/kakkannika/my_portfolio
   ```
2. Navigate to the project directory:
   ```
   cd Portfolio-Project
   ```
3. Open `index.html` in your preferred web browser to view the portfolio.

## Usage Guidelines
- Customize the content in `index.html` to reflect your own projects, skills, and achievements.
- Update the `home.css` file to modify styles according to your design preferences.
- Add images to the `Images` directory as needed to enhance the visual appeal of the portfolio.
